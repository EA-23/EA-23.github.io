Listas HTML 

                 Las Listas HTML permiten a los desarrolladores web agrupar un conjunto de elementos relacionados en: 
                 Lista desordenada <ul> 
                 Lista ordenada <ol> 
                

                Ejemplo: 
                
                        <h3 style="color: red;">Mis pasatiempos</h3>
                        <ol>
                            <li>Leer</li>
                            <li>Hacer trends de Tiktok</li>
                            <li>Escuchar Musica</li>
                        </ol>

                        <h3 style="color: purple;"> Artistas favoritos </h3>
                        <ul>
                            <li>Kenia OS</li>
                            <li>Ariana Grande</li>
                            <li>Shakira</li>
                        </ul>
