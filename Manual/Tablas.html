Tablas HTML 

                 Las tablas HTML <table>  permiten a los desarrolladores web organizar los datos en: 
                 Filas <tr> 
                 Celdas <td> 
                 Encabezado <th> 
                

                Ejemplo: 
                
                        <table>
                            <tr>
                                <th style="background-color: purple; color: white;>Personalidad 1</th>
                                <th style="background-color: blue; color: white;>Personalidad 2</th>
                                <th style="background-color: blueviolet; color: white;>Personalidad 3</th>
                            </tr>
                            <tr>
                                <td>Ever</td>
                                <td>Antonio</td>
                                <td>Eash</td>
                            </tr>
                            <tr>
                                <td> Mi parte femenina</td>
                                <td>Mi parte masculina</td>
                                <td>Una combinacion de las dos</td>
                            </tr>
                        </table>
