Media HTML 

                Multimedia en la web viene en muchos formatos diferentes. 
                Puede ser casi cualquier cosa que pueda escuchar o ver, como
                imágenes, música, sonido, videos, discos, películas, animaciones y más.
                Las páginas web suelen contener elementos multimedia de diferentes tipos y formatos.
                

                Ejemplo:
                
                    <h3>Video HTML</h3>
                    <video width="320" height="240" controls>
                    <source src="Media\2ec77e0d8f9e1c1570e3743aeaf65d89.mp4" type="video/mp4">
                    <source src="movie.ogg" type="video/ogg">
                    </video>

                    <iframe width="560" height="315" 
                    src="https://www.youtube.com/embed/WDxhZXLW8AY" title="YouTube video player" frameborder="0" allow="accelerometer; 
                    autoplay; clipboard-write; encrypted-media; 
                    gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>
                    
                
                Video HTML
                

                
                

                
                    <h3>Audio HTML HTML</h3>
                    <audio controls>
                    <source src="horse.ogg" type="audio/ogg">
                    <source src="Media\Kenia OS - Malas Decisiones (Video Oficial).mp3" type="audio/mpeg">
                    </audio>
